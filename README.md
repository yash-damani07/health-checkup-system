# VitalCheck — Health Checkup System (MERN Stack)

A full-stack health checkup management system built with MongoDB, Express, React (Vite), and Node.js.

## Features
- Patient registration & login (JWT auth)
- Book health checkups (General, Heart, Diabetes, Blood Test, Eye, Dental, Women's Health, etc.)
- Patient dashboard with stats and recent reports
- Cancel pending/confirmed checkups
- Admin/Doctor panel: view all checkups, update status, add test results (BP, heart rate, sugar, cholesterol, BMI, etc.)
- Patient reports page showing full vitals history

See the setup guide below for running this on Windows with VS Code.
